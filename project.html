Projects


Machine Learning with Bank of America

A credit card fraud detection program from trained 284,807 cc transactions from European cardholders from September 2013. This website empowers business owners to upload a record of cc transactions to catch anomalies.

Predicting Wine Quality using Machine Learning

Exploring the relationship between the physicochemical properties of wine (e.g. fixed acidity, volatile acidity, pH, alcohol, chlorides, and others) and the quality of wine. Trained and tested models using UCI ML Repository with the accuracy of 80-85%.

Experimentation and Data Analysis with OpenPose

My capstone project where I studied a open-source library called the OpenPose, which was developed by the CMU Perceptual Computing Lab that is capable of jointly detecting many human bodies in a given input (image, video, webcam). My project consists of performing data analysis of the OpenPose output and studying how occlusions are formed and when occlusions appear the most.

Code for Good with JPMorgan Chase & Co.

A web and mobile application for International Rescue Committee that provides communication portal for fieldworkers in countries undergoing wars, conflicts, and natural disasters. The communication portal includes emergency alert, contact list, local news, check-ins, and messaging.

Data Visualization using New York City 311's Complaint Dataset

A dashboard allowing users to filter the 311 complaint types and time to observe and locate the most problematic neighborhoods in NYC in categories such as parking, garbage, flooding, rodent, etc.
